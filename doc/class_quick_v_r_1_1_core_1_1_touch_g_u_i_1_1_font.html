Initializes the components of the font to the values specified by the strings.

Parameters
FontFaceThe path to the font face to load for this font.
FontSizeThe integer non-zero size of this font.

This method is used primarily in conjunction with XML-based layouts.

Instantiate a font object with a green colour, a font face of Roboto and a font size of 16:

Font f = new Font("rgb(0,255,0)", "fonts/roboto", "16");

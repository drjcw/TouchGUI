Initializes the components of the tab widget to the values specified by the strings.

Parameters
IDThe unique id (if any) of this object.
MaxTabWidthThe maximum (non-zero) width a page tab can be (can be null for unconstrained width)
ParentThe parent layout (used for adding and validating page/widget ids)
ChildrenThe XML node list containing the child elements that compose the style of the button (see XML notation docs for further information).
XThe x positioning of the tab container rect relative to the parent as governed by the horizontal alignment as follows:
Alignment X Position
null or empty x units from parent's left edge
Left x units from parent's left edge
Center x position is ignored
Right x units from parent's right edge
YThe y positioning of the tab container rect relative to the parent as governed by the vertical alignment as follows:
Alignment Y Position
null or empty y units from parent's top edge
Top y units from parent's top edge
Center y position is ignored
Bottom y units from parent's bottom edge
WThe width of the tab container rect.
HThe height of the tab container rect.
HAlignThe horizontal alignemnt of the tab container rect within the parent rect as specified by the following values:
String Value Alignment.Type
null or empty null
left Alignment.Type.Left
center Alignment.Type.Center
right Alignment.Type.Right
VAlignThe horizontal alignemnt of the tab container rect within the parent rect as specified by the following values:
String Value Alignment.Type
null or empty null
top Alignment.Type.Top
center Alignment.Type.Center
bottom Alignment.Type.Bottom

This constructor is used primarily in conjunction with XML-based layouts

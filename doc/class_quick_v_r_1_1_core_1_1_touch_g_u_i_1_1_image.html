Parameters
IDThe unique id (if any) of this object.
SrcThe path (if any) to the image file to load.
ColourThe colour tint to apply to the image or the colour fill to use as the image.
Src Value Colour Value Result
Specified Specified Image is rendered with Colour tint
Specified null or empty Image is rendered without tint
null or empty Specified Colour is used as fill colour for image
null or empty null or empty Invalid, exception thrown
ScaleThe scale mode used for rendering the image to the specicified unit rect dimensions:
String Value Description
null or empty Stretches the texture to fill the complete rectangle passed in to GUI.DrawTexture
stretchToFill Stretches the texture to fill the complete rectangle passed in to GUI.DrawTexture
scaleAndCrop Scales the texture, maintaining aspect ratio, so it completely covers the position rectangle passed to GUI.DrawTexture. If the texture is being draw to a rectangle with a different aspect ratio than the original, the image is cropped
scaleToFit Scales the texture, maintaining aspect ratio, so it completely fits withing the position rectangle passed to GUI.DrawTexture
XThe x positioning of the image relative to the parent as governed by the horizontal alignment as follows:
Alignment X Position
null or empty x units from parent's left edge
Left x units from parent's left edge
Center x position is ignored
Right x units from parent's right edge
YThe y positioning of the image relative to the parent as governed by the vertical alignment as follows:
Alignment Y Position
null or empty y units from parent's top edge
Top y units from parent's top edge
Center y position is ignored
Bottom y units from parent's bottom edge
WThe width of the image as follows:
W H Description
null or empty null or empty The image's dimensions (in pixels) will be used
null or empty Specified The width will be calculated according to the specified height and aspect ration of the image
Specified -n/a- The specified width will be used

H W Description

null or empty Specified The height will be calculated according to the specified width and aspect ration of the image
Specified -n/a- The specified height will be used

Instantiate an image object with the id "myImage", the path "somePic", no colour tint, default scale, a width of 300 virtual pixels, a height calculated automatically, positioned 10 virtual units down and centered horizontally:

Image img = new Image("myImage", "somePic", null, 300, null, null, "10", "center", null);

Instantiate an image object no id, a solid colour fill, default scale, a width of 300 virtual pixels, a height calculated automatically, positioned 10 virtual units down and centered horizontally:

Image img = new Image(null, null, "rgb(255,0,255)", 300, null, null, "10", "center", null);

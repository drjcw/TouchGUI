Detailed Description

Specifies a formatted text for use in text rendering.

Initializes the components of the text to the values specified by the strings.

Parameters
TextThe text string to be rendered.
TextAlignThe string containing the text alignment.
Format Description
null or empty Aligns the text to the upper-left corner of the containing element.
upperLeft Aligns the text to the upper-left corner of the containing element.
upperCenter Aligns the text to the center of the top edge of the containing element.
upperRight Aligns the text to the upper-right corner of the containing element.
middleLeft Aligns the text to the center of the left edge of the containing element.
middleCenter Aligns the text to the center of the containing element.
middleRight Aligns the text to the center of the right edge of the containing element.
lowerLeft Aligns the text to the lower-left corner of the containing element.
lowerCenter Aligns the text to the center of the bottom edge of the containing element.
lowerRight Aligns the text to the lower-right corner of the containing element.
WordWrapSpecifies whether or not the text will be wrapped within the containing element.
Format Description
null or empty No word wrap will be applied
false No word wrap will be applied
true Word wrap will be applied
TextColourThe string containing the text colour.
Format Description Example
rgb Specifies a colour in rgb format rgb(255,0,0)
rgba Specifies a colour in rgb format with an alpha component rgba(255,0,0,128)
FontFaceThe path to the font face to load for this text.
FontSizeThe integer non-zero size of this text.

This method is used primarily in conjunction with XML-based layouts.

Instantiate a text object for the string "Hello World!" with centered text alignment, no word wrap, a green colour, a font face of Roboto and a font size of 16:

Text t = new Text("Hello World!", "middleCenter", null, "rgb(0,255,0)", "fonts/roboto", "16");

Calculates the bounding rect (in absolute pixel coordinates) of the text object based on the text string contents, text formatting style and container unit rect.

Parameters
VirtualScaleThe scaling factor used to map virtual units to pixel units if component unit type is virtual.
ParentThe parent rect (in pixel units) if component unit type is percentage.
ContainerThe container unit rect of the element this text object belongs to.
Returns
The bounding rect (in absolute pixel coordinates) of this text object.

Note: if either the width or height of the container are null, the size of the bounding rect will be that of the container.

Calculate the absolute pixel bounding rect for a label widget within the specified parent rect:

UnityEngine.Rect rPx = text.CalculatePixelRect(VirtualScale, Parent, rect);

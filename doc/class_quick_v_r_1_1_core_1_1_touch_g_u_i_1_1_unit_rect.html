Parameters

Alignment X Position
null or empty x units from parent's left edge
Left x units from parent's left edge
Center x position is ignored
Right x units from parent's right edge

Alignment Y Position

Top y units from parent's top edge
Center y position is ignored
Bottom y units from parent's bottom edge
WThe width of the unit rect.
HThe height of the unit rect.
HAlignThe horizontal alignemnt of the unit rect within the parent rect as specified by the following values:
String Value Alignment.Type
null or empty null

VAlignThe horizontal alignemnt of the unit rect within the parent rect as specified by the following values:
String Value Alignment.Type
null or empty null
top Alignment.Type.Top
center Alignment.Type.Center
bottom Alignment.Type.Bottom

This constructor is used primarily in conjunction with XML-based layouts

Instantiate a unit rect object spaced 10% from the left edge of the parent rect, measuring 100x100 virtual units and centered vertically:

UnitRect r = new UnitRect("10%", null, "100", "100", null, "center");

Parameters
IDThe unique id (if any) of this object.
TextThe text string to be rendered.
TextAlignThe string containing the text alignment.
Format Description
null or empty Aligns the text to the upper-left corner of the containing element.
upperLeft Aligns the text to the upper-left corner of the containing element.
upperCenter Aligns the text to the center of the top edge of the containing element.
upperRight Aligns the text to the upper-right corner of the containing element.
middleLeft Aligns the text to the center of the left edge of the containing element.
middleCenter Aligns the text to the center of the containing element.
middleRight Aligns the text to the center of the right edge of the containing element.
lowerLeft Aligns the text to the lower-left corner of the containing element.
lowerCenter Aligns the text to the center of the bottom edge of the containing element.
lowerRight Aligns the text to the lower-right corner of the containing element.
WordWrapSpecifies whether or not the text will be wrapped within the containing element. Note: word wrapping can only be enabled if the label has a defined width.
Format Description
null or empty No word wrap will be applied
false No word wrap will be applied
true Word wrap will be applied
TextColourThe string containing the text colour.
Format Description Example
rgb Specifies a colour in rgb format rgb(255,0,0)
rgba Specifies a colour in rgb format with an alpha component rgba(255,0,0,128)
FontFaceThe path to the font face to load for this text.
FontSizeThe integer non-zero size of this text.
XThe x positioning of the label relative to the parent as governed by the horizontal alignment as follows:
Alignment X Position
null or empty x units from parent's left edge
Left x units from parent's left edge
Center x position is ignored
Right x units from parent's right edge
YThe y positioning of the label relative to the parent as governed by the vertical alignment as follows:
Alignment Y Position
null or empty y units from parent's top edge
Top y units from parent's top edge
Center y position is ignored
Bottom y units from parent's bottom edge
WThe width of the label.
W Value WordWrap Value Calculated Width
null or empty false Width of string according to text style
null or empty true Invalid, exception thrown
Specified false Width of string according to specified width
Specified true Width of string according to specified width
HThe height of the label.
H Value Calculated Height
null or empty Height of string according to text style
Specified Height of string according to specified height
HAlignThe horizontal alignemnt of the label within the parent rect as specified by the following values:
String Value Alignment.Type
null or empty null
left Alignment.Type.Left
center Alignment.Type.Center
right Alignment.Type.Right
VAlignThe horizontal alignemnt of the label within the parent rect as specified by the following values:
String Value Alignment.Type
null or empty null
top Alignment.Type.Top
center Alignment.Type.Center
bottom Alignment.Type.Bottom

This constructor is used primarily in conjunction with XML-based layouts

Instantiate a label object with an id of "myString", the text value of "Hello World!" with centered text alignment, no word wrap, a green colour, a font face of Roboto and a font size of 16 spaced 10% from the left edge of the parent rect, measuring 100x100 virtual units and centered vertically:

Label label = new Label("myString", "Hello World!", "middleCenter", null, "rgb(0,255,0)", "fonts/roboto", "16", "10%", null, "100", "100", null, "center");

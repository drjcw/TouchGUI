static UnityEngine.Color QuickVR.Core.TouchGUI.Colour.Parse ( string  Colour)
static

Parses a colour string and returns a colour object.

Parameters
ColourThe string containing the colour information.
Format Description Example
rgb Specifies a colour in rgb format rgb(255,0,0)
rgba Specifies a colour in rgb format with an alpha component rgba(255,0,0,128)
Returns
The colour object initialized to colour information specified in Colour .

This method is used primarily in conjunction with XML-based layouts.

Generate a colour object from a string containing a red colour:

UnityEngine.Color c = Colour.Parse("rgb(255,0,0)");

Generate a colour object from a string containing a red colour with half transparency:

UnityEngine.Color c = Colour.Parse("rgba(255,0,0,128)");
